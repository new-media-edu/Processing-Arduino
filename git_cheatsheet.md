# Git Cheat Sheet

## Push changes to repo (upload)
1. cd ...
2. git add .
3. git commit -m "whatever"
4. git push origin master

## Pull changes from repo (download)
1. Goto the repo URL you want to download
2. Look for "Clone or Download" button and click it
3. copy URL
1. cd ... (cd into the directory you want to download into)
2. git clone <paste that URL>
